# 开发文档

这套文档按模块拆分，不再用单个大文档承载全部内容。

## 结构

- `websocket/`：WebSocket 协议、会话、测试
- `jsonrpc/`：JSON-RPC 协议、服务器封装、测试
- `build/`：构建、链接、测试集成
- `examples/`：示例程序说明

## 使用顺序

1. 先看 `websocket/`
2. 再看 `jsonrpc/`
3. 再看 `build/`
4. 参考 `examples/` 运行示例程序

## 参考关系

- `docs/refactor/`：约束、迁移、验收
- `docs/develop/`：落地实现说明
